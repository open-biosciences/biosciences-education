# biosciences-education

Training materials, tutorials, and onboarding guides for the Open Biosciences platform.

Part of the [Open Biosciences](https://github.com/open-biosciences) multi-repo platform.

## Status

**Planned.** Content will be authored after the core platform migration (Waves 1-3) is complete. This repo depends on stable APIs and workflows being in place before training materials can be written.

## What Will Be Here

- Tutorial notebooks demonstrating platform capabilities
- Onboarding checklist for new contributors
- Training materials for using MCP tools, knowledge graph queries, and agent workflows
- Guided walkthroughs of key research workflows
